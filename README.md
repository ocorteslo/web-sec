# Web Security Course / Curso de Seguridad Web

Este repositorio contiene el código, ejemplos y recursos para el curso de seguridad web.

## Estructura del Curso

### 1. Conceptos Básicos
- Introducción a la seguridad web
- Modelo de seguridad del navegador
- HTTPS y certificados

### 2. Vulnerabilidades Comunes (OWASP Top 10)
- Injection (SQL, NoSQL, Command)
- Broken Authentication
- Sensitive Data Exposure
- XML External Entities (XXE)
- Broken Access Control
- Security Misconfiguration
- Cross-Site Scripting (XSS)
- Insecure Deserialization
- Using Components with Known Vulnerabilities
- Insufficient Logging & Monitoring

### 3. Prácticas de Seguridad
- Validación de entrada
- Codificación de salida
- Gestión de sesiones
- Control de acceso
- Configuración segura

## Estructura del Repositorio

## Requisitos

- Conocimientos básicos de desarrollo web (HTML, CSS, JavaScript)
- Node.js (para ejemplos prácticos)
- Un navegador web moderno

## Uso

Cada carpeta contiene ejemplos específicos con instrucciones de uso en sus respectivos README.
