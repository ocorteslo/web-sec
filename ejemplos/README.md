# Ejemplos de Seguridad Web

Esta carpeta contiene ejemplos prácticos de vulnerabilidades web comunes y sus soluciones.

## Contenido

### XSS (Cross-Site Scripting)
- Ejemplos de XSS reflejado, almacenado y DOM-based
- Técnicas de sanitización y prevención

### SQL Injection
- Ejemplos de inyección SQL básica y avanzada
- Uso de consultas parametrizadas y ORM

### CSRF (Cross-Site Request Forgery)
- Demostración de ataques CSRF
- Implementación de tokens anti-CSRF

### Autenticación y Sesiones
- Gestión segura de sesiones
- Almacenamiento seguro de contraseñas (hashing)

## Cómo usar los ejemplos

Cada subdirectorio contiene:
- `vulnerable/` - Código con vulnerabilidades
- `secure/` - Código corregido y seguro
- `README.md` - Explicación detallada de la vulnerabilidad y la solución
