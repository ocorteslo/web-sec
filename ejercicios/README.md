# Ejercicios Prácticos

Esta carpeta contiene ejercicios para practicar los conceptos de seguridad web.

## Lista de Ejercicios

### Nivel Básico
1. Identificar y corregir vulnerabilidades XSS
2. Prevenir inyección SQL
3. Implementar autenticación segura

### Nivel Intermedio
4. Configurar CORS correctamente
5. Implementar Content Security Policy (CSP)
6. Gestión segura de tokens JWT

### Nivel Avanzado
7. Análisis de seguridad de aplicaciones completas
8. Implementar rate limiting
9. Configuración de headers de seguridad

## Instrucciones

Cada ejercicio incluye:
- Descripción del problema
- Código inicial
- Tests para verificar la solución
- Solución de referencia (en carpeta separada)
