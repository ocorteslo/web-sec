- **SQLi**: SQL Injection
- **CSP**: Content Security Policy
- **CORS**: Cross-Origin Resource Sharing
- **JWT**: JSON Web Token
- **HTTPS**: HyperText Transfer Protocol Secure
- **TLS**: Transport Layer Security
